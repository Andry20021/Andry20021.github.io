Projects

Halloween Hangman

This Halloween-themed Hangman game utilizes Node.js, Express.js, and Handlebars for dynamic rendering. Express.js manages routes for actions like starting a new game and selecting letters, while Handlebars updates the display of remaining guesses and hangman images as players make their selections.

During development, Nodemon automatically restarts the server with each code change, making it easier to test and adjust. This setup ensures smooth gameplay with real-time updates, enhancing the overall user experience.
